How to use this prototype

• Clickable areas are indicated with a yellow hover state. This hover state is for your convenience only - there will be no hover state in the real experience.

• You can see the effect of the filters by paying attention to whether or not a content block is an App or a Website, and the numbers for each row.

 

 

Open filter panel - Expected behavior

• Clicking "Change View" button slides Filter Content panel up from bottom (main page content remains below this panel).

• If no filters have been applied, "All types" and "Your device" and "Worldwide+Your Country" are the default selections.

• If filters have been applied, the respective Device, Type, and Country bar contains the names of the selected filters. Note that Country selections are generalized, not specific here.

• Clicking "Cancel" closes Filter Content panel (slides down) and preserves existing selections on the screen where Filter Content panel was invoked (no changes applied).

• Clicking "Done" closes Filter Content Panel (slides down) and applies changed selections to the content on the screen where "Filter Content" panel was invoked.

 

Filter Device / Type / Sort by / Display as bars - Expected Behavior

• Clicking a bar expands to show options and closes other options if they were open.

 

Filter Country bar - Expected Behavior

• Clicking "Country" bar slides Country options in from right (main page slides out of view to left)..

 

• Filter By Device / Type / Sort by / Display as panels - Expected Behavior

• The current selected filter is disabled (not clickable/hoverable).

• Clicking any active item selects it.

• Can only select one device type at a time (radio buttons).

 

Filter By Country Options panel - Expected Behavior

• Can multi-select country options. Only Worldwide and Brazil are clickable in this prototype.

• Clicking "back" arrow saves the new selections and updates the Country bar copy.

• Note that new selections are not applied until user clicks "Done" on the Filter Content panel. So if user makes new selections here, then cancels on Filter Content, the current Country selections are still applied.
