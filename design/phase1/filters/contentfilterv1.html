How to use this prototype

• Clickable areas are indicated with a yellow hover state. This hover state is for your convenience only - there will be no hover state in the real experience.

• You can see the effect of the filters by paying attention to whether or not a content block is an App or a Website, and the numbers for each row.

• Popular is not a clickable option in this prototype.

• List/Grid is not a clickable option in this prototype.

 

 

Filter Content panel - Expected behavior

• Clicking "Change View Option" button slides Filter Content panel up from bottom (main page content remains below this panel).

• If no filters have been applied, "Your device" and "All types" and "New" and "List" are the default selections.

• If filters have been applied, the selected option is highlighted in the Device, Type, and Sort sections.

• Clicking "Reset" resets options to their default state. User clicks "Done" to close the panel and apply the reset filters.

• Clicking "Done" closes Filter Content Panel (slides down) and applies changed selections to the content on the screen where "Filter Content" panel was invoked.

 

• Filter By Device / Content / Sort by / Display - Expected Behavior

• Can only select one option at a time.

 

 

Settings panel - expected behavior

• Default Country Selection is my detected country + Worldwide.

• Clicking the Current Country Selection bar opens Country Selection panel (slides in from right, while Settings panel slides out to left).

• Clicking back arrow returns user to Filter Content panel (Filter Content slides in from left, while Settings panel slides out to right).

 

 

Country Selection panel - expected behavior

• Defaults are my detected country + Worldwide.

• User can select as many countries as desired.

• Clicking back arrow returns user to Settings panel (slides in from left, while Country Selection slides out to right).

• Changes made to country selection are reflected in the Current Country Selection bar on the Settings panel.
